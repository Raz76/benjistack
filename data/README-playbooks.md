# BenjiStack Business-Type Playbooks

This folder contains the recommendation backbone for the new PDF system.

## Files
- `business-type-playbooks.json` — implementation-ready recommendation data by business type and phase

## What it does
For each business type, it defines:
- summary
- fastest path to revenue
- Launch / Prepare / Grow objectives
- success checkpoints
- common mistakes to avoid
- tool recommendations
- Launch-phase path choices:
  - `best_of_breed`
  - `all_in_one`

## Why this matters
This lets BenjiStack generate a PDF that feels personalized without writing every report from scratch.

The PDF can:
- classify the idea into a small number of business types
- reuse the same core structure
- insert business-type-specific recommendations
- recommend GHL naturally when an all-in-one setup genuinely fits

## Recommendation philosophy
Trust first.

The report should:
- explain why a tool fits
- show tradeoffs when useful
- avoid sounding forced or overly affiliate-driven
- use GHL where consolidation is genuinely helpful, especially in Launch
